escape sequence characters :
	\n	adds a new line
	\t	adds horizontal tab space
	\f	form feed rather a new page
	\b	moves cursor one character back
	\a	adds a beep (alert sound)
	\r	adds carraige return
	\\	prints only \
	\"
	\'

Format specifiers:
	%d		decimal integer
	%c		character
	%f		float
	%x		hexadecimal
	%o		octal
	%s		string	
	%u		unsigned
	%lf		long float
	%%		to print actually %
 	
  
Variable Naming Rule:

  - Can not add space or use any special character
  - Using Alphabets,digits and _(underscore) is allowed
  - Can not start identifier name with number/digit
